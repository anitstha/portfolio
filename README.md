# Portfolio

A minimalist portfolio website for Anit Shrestha, Frontend Developer.

## Tech Stack

- React
- React Router DOM
- Tailwind CSS
- Vite

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Project Structure

```
src/
├── components/     # Reusable UI components
├── constants/      # Data files (personal info, skills, projects)
├── layouts/        # Page layouts
└── pages/          # Route pages
```
